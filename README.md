# LoCyanFrpPanel

LoCyanFrp Panel, Powered by NaiveUI

## 地址

新面板地址: [https://preview.locyanfrp.cn](https://preview.locyanfrp.cn)

## 如何安装/运行

1. clone本仓库

```shell
git clone https://github.com/LoCyan-Team/LoCyanFrpPanel.git
```

2. 安装Nodejs 16x/18x，并运行命令安装

```shell
npm i
```

3. 运行

```shell
npm run dev
```

4. 访问输出的地址，即可访问!
