# LoCyanFrpPanel

LoCyanFrp新面板, By NaiveUI

## Live

新面板地址: [https://preview.locyanfrp.cn](https://preview.locyanfrp.cn)

## How to install/run

clone本仓库
```shell
git clone https://github.com/LoCyan-Team/LoCyanFrpPanel.git
```

安装Nodejs 16x/18x，并运行命令安装
```shell
npm i
```

运行
```shell
npm run dev
```

访问输出的地址，即可访问!
