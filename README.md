## 如何使用

1. clone 本仓库

```shell
git clone https://github.com/LoCyan-Team/LoCyanFrpPanel.git
```

2. 安装 [Node.js 18+](https://nodejs.org)

3. 安装 yarn

```shell
npm i -g yarn
```

4. 安装依赖

```shell
yarn install
```

5. 运行项目

```shell
yarn dev
```
